# Phase 5: 디자인

## 최종 선택: 감정적 따뜻함 (코랄 핑크 + 부부 중심)

**선택 근거**:
- 색상 톤: 코랄 핑크 (`#ff6b6b`) + 따뜻한 크림 배경
- 철학: 부부가 함께 절약하는 **감정적 연결** 강화
- 레이아웃: 큰 카드 + 넉넉한 여백 (진입 장벽 낮음)
- 시각화: 이모지로 카테고리 직관화, 원형 Donut 진행률
- 페르소나 매칭: 영희(감정 중심)에 완벽, 철수도 명확한 정보 계층으로 만족

---

## 디자인 철학

**"❤️ 우리 함께 절약 중이에요"**

1. **감정적 연결**: 부부가 함께하는 경험으로 재정 관리를 따뜻하게 표현
2. **카드 중심 레이아웃**: 각 정보를 독립적 카드로 분리 → 명확성 + 따뜻함의 균형
3. **친근한 이모지**: 숫자와 텍스트만 있는 것이 아니라 감정적 언어 추가
4. **원형 진행률**: 도넛 차트로 목표 달성도를 한눈에 시각화
5. **넉넉한 여백**: 숨을 쉴 수 있는 공간으로 편안함 제공

### 핵심 레이아웃 원칙
1. **카드 기반**: 정보를 박스로 분리 → 한눈에 스캔 가능
2. **여백 우선**: 16px~20px 패딩으로 숨을 쉴 수 있는 공간
3. **세로 흐름**: 모바일 우선 단일 컬럼
4. **중앙 정렬**: 태블릿 이상에서 max-width 500px

## 컴포넌트 명세

### 1. Header
```css
배경: linear-gradient(135deg, #ff8a80, #ff6b6b)
높이: 100px ~ 120px
패딩: 24px (좌우), 20px (상하)
border-radius: 0 0 16px 16px
box-shadow: 0 2px 8px rgba(255, 107, 107, 0.2)
```
**내용**:
- 이모지: 28px (❤️)
- 제목: "우리 함께 절약 중이에요" (13px Bold)
- 날짜: "2026년 7월 3일, 목요일" (11px 보조)

---

### 2. Progress Circle (Donut)
```css
크기: 120px × 120px
배경: conic-gradient(#ffd4a3 65%, #f0f0f0 35%)
border-radius: 50%
box-shadow: 0 4px 12px rgba(255, 107, 107, 0.15)

내부 구성:
  - 바깥쪽: 색칠된 부분 (금색, 65%)
  - 안쪽: 흰색 원 (Donut 효과)
  - 중앙 텍스트: "65%" (24px Bold, #ff6b6b)
```
**위치**: 헤더 아래 중앙, margin: 20px auto

---

## 마이크로 애니메이션

### 버튼 클릭
```css
transition: all 0.2s cubic-bezier(0.34, 1.56, 0.64, 1)
/* scale: 1 → 0.98 → 1 (스프링 효과) */
```

### 카드 탭
```css
transition: all 0.2s ease
/* box-shadow, transform 부드럽게 */
```

### 진행률 애니메이션 (페이지 로드)
```
0% → 진행률 빈 상태
1.5s → 최종 퍼센트로 채워짐 (ease-out)
```

## 접근성 (Accessibility)

### 색상 대비 (WCAG)
```
- 검정색 (#333) on 흰색: 명도비 12:1 ✓✓✓
- 흰색 on 코랄 (#ff6b6b): 명도비 3.9:1 ✓✓
- 회색 (#666) on 흰색: 명도비 4.5:1 ✓✓
```

### 터치 타겟
```
최소 크기: 48px × 48px (버튼, 탭)
최소 간격: 8px
```

### 키보드 네비게이션
```
- Tab: 모든 상호작용 요소 이동
- Enter: 버튼 활성화
- 포커스: 2px solid #ff6b6b 테두리
```

### 포커스 스타일
```css
*:focus {
  outline: 2px solid #ff6b6b;
  outline-offset: 2px;
}
```
