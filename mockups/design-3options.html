🎨 가계부 앱 디자인 - 3가지 시안

시안 1️⃣ 투명한 신비로움 (추천)

특징: 극도로 투명한 글래스 + 차분한 파랑 톤

색상: 진한 파랑/검정 배경 + 파란 톤 강조

레이아웃: 목표 원형 크게 + 카테고리 카드 아래

인터랙션: 부드러운 호버, 미묘한 글로우

✓ 장점: 세련되고 현대적, 눈 편함, 차별성 높음

✗ 약점: 진지한 톤, 따뜻함 부족

타깃: 철수 (기술친화적, 미니멀 선호)

시안 2️⃣ 극도의 미니멀

특징: 검정-흰색-회색만. 데이터 중심

색상: #000000, #FFFFFF, #808080만 사용

레이아웃: 그리드 중심, 균등한 공간 배분

인터랙션: 클릭 시 반전 또는 테두리 강조

✓ 장점: 가장 빠름, 명확함, 인쇄 호환성

✗ 약점: 차별성 없음, 따뜻함 부족, 지루할 수 있음

타깃: 데이터 분석가, 미니멀 매니아

시안 3️⃣ 따뜻한 집

특징: 밤하늘 어두운 톤 + 오렌지/골드 강조

색상: 따뜻한 갈색/검정 배경 + 골드/오렌지 강조

레이아웃: 따뜻한 카드형, 둥근 모서리

인터랙션: 골드 글로우, 부드러운 애니메이션

✓ 장점: 친근하고 따뜻함, 가족 중심, 감정적 연결

✗ 약점: 진부할 수 있음, 세련됨 부족

타깃: 영희 (감정 중심, 따뜻함 선호)
